Q71. Using Java's Reflection API, you can use _ to get the name of a class and _ to retrieve an array of its methods.

  • this.getClass().getSimpleName() ‚Üí Returns the simple name of the class (without the package).
  • Alternatively, this.getClass().getName() returns the fully qualified name (including the package).
  • this.getClass().getDeclaredMethods() ‚Üí Returns an array of Method objects representing all declared methods (including private/protected ones, but excluding inherited methods unless they are overridden).
  • If you want all public methods (including inherited ones), you could use getMethods() instead.
Q72. Which is not a valid lambda expression?

  • Lambda expressions in Java require parentheses around parameters if the parameter type is explicitly declared or if there are multiple parameters.
  • Valid lambda expressions:
    • a -> false; ‚Üí Single parameter, no type, parentheses optional.
    • (a) -> false; ‚Üí Single parameter, no type, parentheses allowed (but optional).
    • (String a) -> false; ‚Üí Single parameter with type, parentheses required.
