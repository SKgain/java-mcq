Q33. Which letters will print when this code is run? 
public static void main(String[] args) {
    try {
        System.out.println("A");
        badMethod();
        System.out.println("B");
    } catch (Exception ex) {
        System.out.println("C");
    } finally {
        System.out.println("D");
    }
}
public static void badMethod() {
    throw new Error();
}

  • Error is not inherited from Exception, so it is not caught by the catch(Exception ex) block.
  • The output prints "A" and then the finally block prints "D".
  • System.out.println("B"); is never executed because the badMethod() throws an Error.

Q35. What is the output of this code? 
import java.util.*;
class Main {
    public static void main(String[] args) {
        String[] array = {"abc", "2", "10", "0"};
        List list = Arrays.asList(array);
        Collections.sort(list);
        System.out.println(Arrays.toString(array));
    }
}

  • The Arrays.asList(T... a) method returns a fixed-size list backed by the specified array.
  • Sorting the list changes the original array since the list is backed by the array.
  • Sorting is lexicographical (string order), so the order becomes: "0", "10", "2", "abc".

Q38. Which approach cannot be used to iterate over a List named theList?
A 
for (int i = 0; i < theList.size(); i++) {
    System.out.println(theList.get(i));
}
B 
for (Object object : theList) {
    System.out.println(object);
}
C 
Iterator it = theList.iterator();
for (it.hasNext()) {
    System.out.println(it.next());
}
D 
theList.forEach(System.out::println);

The syntax for (it.hasNext()) is invalid. It should be a while loop: while (it.hasNext()).
