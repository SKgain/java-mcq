Q11. What is the result of this code? 
class MainClass {
    final String message() {
        return "Hello!";
    }
}

class Main extends MainClass {
    public static void main(String[] args) {
        System.out.println(message());
    }

     String message() {
         return "World!";
     }
 }

Compilation error at line 10 because final methods cannot be overridden, and here message() is a final method. Also, note that the non-static method message() cannot be referenced from a static context in the main method.

Q13. What is the output of this code? 
class Main {
    public static void main(String[] args){
        int a = 123451234512345;
        System.out.println(a);
    }
}

The int type in Java can represent whole numbers from -2,147,483,648 to 2,147,483,647. The value 123451234512345 exceeds this range, so the code will not compile.
