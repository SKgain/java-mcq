  • map.entrySet() returns a Set<Entry<K, V>> (where Entry is a nested interface in Map).
  • To iterate over key-value pairs, we use:
    • for (Map.Entry<String, Integer> entry : map.entrySet())
  • Map.Entry<String, Integer> represents a key-value pair.

Q158. What will be the output of the following Java program? 
class variable_scope {
    public static void main(String args[]) {
        int x;
        x = 5;
        {
            int y = 6;
            System.out.print(x + " " + y);
        }
        System.out.println(x + " " + y);
    }
}

  • The variable y is declared within a block scope.
  • Outside of this block, y is no longer available.
  • Accessing y outside of its scope results in a compilation error.
