Q121. Which is the correct return type for the processFunction method? 
_____ processFunction(Integer number, Function<Integer, String> lambda) {
    return lambda.apply(number);
}

  • The method lambda.apply(number) returns a String based on the Function<Integer, String> interface.
  • Therefore, the method return type must be String.
  • The correct method signature would be: String processFunction(Integer number, Function<Integer, String> lambda)

Q122. What function could you use to replace slashes for dashes in a list of dates? 
List<String> dates = new ArrayList<>();
// missing code
dates.replaceAll(replaceSlashes);

  • The replaceAll method on a list takes a UnaryOperator<T> to transform each element in-place.
  • UnaryOperator is a specialization of Function<T, T> that returns the same type it receives.
  • Hence, UnaryOperator<String> replaceSlashes = date -> date.replace("/", "-"); is correct.
Q123. Which functional interface is most appropriate for a lambda that consumes two values and returns nothing?

  • BiConsumer<T, U> is a functional interface that accepts two input arguments and returns no result.
  • It is used for operations that consume two values (e.g., logging, printing, modifying external state).
  • Other options either return a value or accept only one input.
Q124. What is the result of the following code? 
Stream.of("java", "python", "c")
  .sorted((a, b) -> b.compareTo(a))
  .forEach(System.out::print);

  • Stream.of("java", "python", "c") creates a stream of strings.
  • sorted((a, b) -> b.compareTo(a)) sorts the elements in reverse (descending) order.
  • forEach(System.out::print) prints the elements without space.
  • Descending order: "python", "java", "c" → Result: pythonjavac
Q125. What is the result of this stream operation? 
Stream.of(3, 1, 4)
  .map(i -> i * 2)
  .filter(i -> i > 4)
  .forEach(System.out::print);

  • Stream.of(3, 1, 4) → elements: 3, 1, 4
  • .map(i -> i * 2) → becomes: 6, 2, 8
  • .filter(i -> i > 4) → filter: 6, 8
  • Output: 68
Q126. Which functional interface is used to check a condition and return a boolean?

  • Predicate<T> is used to test a condition and return true or false.
  • Commonly used in filter and if conditions within streams.
Q127. What does the Supplier functional interface do?

  • Supplier<T> is a functional interface that supplies a result of type T.
  • It takes no parameters and is often used for lazy value generation.
Q128. What is the output of the following code? 
List<String> list = Arrays.asList("A", "B", "C");
list.stream().limit(2).forEach(System.out::print);

  • limit(2) limits the stream to first 2 elements: A, B
  • forEach prints them → Output: AB
Q129. Which method is used to transform stream elements?

  • map() is used to transform each element of a stream.
  • Other methods serve different purposes: filter for conditional filtering, forEach for processing, and collect for aggregation.
Q130. What does Optional.ofNullable(value) do?

  • Optional.ofNullable(value) creates an Optional that may or may not contain a non-null value.
  • If value is null, returns Optional.empty().
  • It avoids NullPointerException.
