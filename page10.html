Q91. What will this code print, assuming it is inside the main method of a class?
System.out.println("hello my friends".split(" ")[0]);

  • The split(" ") method splits the string into an array using space as delimiter.
  • The first word, at index 0, is "hello".
  • So split(" ")[0] returns "hello".
Q92. You have an instance of type Map<String, Integer> named instruments containing the following key-value pairs: guitar=1200, cello=3000, and drum=2000. If you add the new key-value pair cello=4500 to the Map using the put method, how many elements do you have in the Map when you call instruments.size()?

  • Adding a key that already exists replaces the previous value.
  • So, the number of elements remains 3 since cello was replaced.
Q93. Which class acts as the root class for the Java Exception hierarchy?

  • Throwable is the superclass of all errors and exceptions in Java.
  • It is the root of the Java exception hierarchy.
Q94. Which class does not implement the java.util.Collection interface?

  • HashMap implements the Map interface, not Collection.
Q95. You have a variable named employees of type List<Employee> containing multiple entries. The Employee type has a method getName() that returns the employee name. Which statement properly extracts a list of employee names?

  • The stream() method creates a stream from the list.
  • map(Employee::getName) extracts the names.
  • collect(Collectors.toList()) collects the result back into a list.
Q96. What is the output of this code snippet?
String s1 = "Java";
String s2 = new String("Java");
System.out.println(s1 == s2);

  • == compares object references, not content.
  • s1 and s2 refer to different objects, so result is false.
Q97. What is the default value of a boolean variable declared as an instance field in Java?

  • Instance boolean fields default to false.
  • Local variables must be initialized explicitly.
Q98. What keyword is used to inherit a class in Java?

  • extends keyword is used for class inheritance.
  • implements is used for interfaces.
Q99. Which of the following is NOT a valid way to create a thread in Java?

  • ThreadGroup is used to group threads, not create them.
  • Threads are created by extending Thread or implementing Runnable.
Q100. Which Java keyword is used to prevent method overriding?

  • The final keyword prevents method overriding in subclasses.
  • This helps maintain the method's behavior.
