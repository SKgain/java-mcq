Q56. Declare a variable that holds the first four digits of Π.

  • The default floating-point type in Java is double.
  • float literals require a suffix F to be valid.
  • double pi = 3.141; is correct and accurate for this case.
  • Note: decimal is not a valid type in Java.
  • int cannot store fractional values.
public class TestReal {
    public static void main(String[] args) {
        double pi = 3.14159265;
        float pi2 = 3.141F;
        System.out.println("Pi=" + pi);
        System.out.println("Pi2=" + pi2);
    }
}
Q57. Use the magic power to cast a spell 
public class MagicPower {
    void castSpell(String spell) {}
}

  • To invoke a method in Java, you must first instantiate the object unless it's static.
  • castSpell is a non-static method, so it must be called on an instance of MagicPower.
  • new MagicPower().castSpell("expecto patronum"); creates an object and immediately calls the method.

Q59. What does this code print?

    int temp=a;
    a=b;
    b=temp;
}
