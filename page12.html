Q111. What will this code print?

List<String> list1 = new ArrayList<>();
list1.add("One");
list1.add("Two");
list1.add("Three");

List<String> list2 = new ArrayList<>();
list2.add("Two");

list1.remove(list2);
System.out.println(list1);

  • list1.remove(list2); tries to remove the object list2 from list1, not the contents of list2.
  • Since list2 is not in list1, the call has no effect.
  • Therefore, the original list remains unchanged: [One, Two, Three].
