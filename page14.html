Q131. What values for x and y will cause this code to print "btc"? 
String buy = "bitcoin";
System.out.println(buy.substring(x, x+1) + buy.substring(y, y+2))

  • buy.substring(0,1) gives "b"
  • buy.substring(2,4) gives "tc"
  • So result is "btc"

Q134. What is the output of the following code snippet? 
int a = 5;
int b = 2;
System.out.println(a / b);

  • Both a and b are integers.
  • Integer division discards the decimal part.
  • So, 5 / 2 results in 2.
Q135. Which Java keyword is used to inherit a class?

  • Java uses the extends keyword for class inheritance.
  • Example: class Child extends Parent.
  • implements is used for interfaces.
Q136. What is the purpose of the `static` keyword in Java?

  • Static members are shared among all instances of the class.
  • They can be accessed without creating an object of the class.
  • Commonly used for utility methods or constants.
Q137. What is the result of the expression: true || false && false?

  • AND (&&) has higher precedence than OR (||).
  • false && false is false.
  • true || false is true.
  • Final result: true.
Q138. Which of these is not a valid Java data type?

  • real is not a valid data type in Java.
  • Java uses float and double for real numbers.
  • byte, boolean, and float are all valid types.
Q139. What is the purpose of the final keyword in Java?

  • final variables cannot be reassigned.
  • final methods cannot be overridden.
  • final classes cannot be extended.
Q140. Which collection class allows you to access elements using a key?

  • HashMap stores data in key-value pairs.
  • Elements can be accessed quickly using their keys.
  • Other collections like ArrayList, LinkedList, and HashSet do not use keys.
