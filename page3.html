Q21. How many times will this code print "Hello World!"? 
class Main {
    public static void main(String[] args){
        for (int i=0; i<10; i=i++){
            i+=1;
            System.out.println("Hello World!");
        }
    }
}

  • The loop increment is i = i++, which does not increment i as expected.
  • The expression i++ returns the original value of i, so the assignment i = i++ leaves i unchanged.
  • Inside the loop, i += 1 increments i by 1 each iteration.
  • Effectively, the loop increments i by 1 per iteration until i < 10.
  • Therefore, it prints "Hello World!" exactly 10 times.

Q23. What code would you use in Constructor A to call Constructor B? 
public class Jedi {
  /* Constructor A */
  Jedi(String name, String species){}

  /* Constructor B */
  Jedi(String name, String species, boolean followsTheDarkSide){}
}

  • In Java, you use this(...) to call another constructor within the same class.
  • this(name, species, false) calls Constructor B from Constructor A.
  • Using new Jedi(...) creates a new instance and is not used to chain constructors.
  • super(...) calls a constructor in the superclass, which is not applicable here.
  • Note: The provided code snippet may not compile fully due to missing constructor bodies.
